# **App Name**: Sarathi Connect

## Core Features:

- Citizen Home Portal: Displays a clean, government-style hero section with quick action cards for key services like Apply for Learner/Driving Licence, Renew Licence, Track Application, and Book Driving Test. Includes a multi-language selector and citizen login.
- Personalized Citizen Dashboard: Provides a card-based dashboard for logged-in citizens, showcasing My Applications, Driving Licence Status, Upcoming Test Dates, Uploaded Documents, and Notifications with status badges.
- Guided Application Workflow: A step-based wizard UI for licence applications, guiding users through Personal Information, Document Upload, Fee Payment, Slot Booking, and Confirmation with a visible progress bar.
- Driving Test Slot Booking: Offers a calendar-style UI for booking driving test slots, displaying available times and nearest RTO locations using modern cards.
- Visual Application Tracker: A visual timeline page illustrating the various stages of a licence application, from Submitted to Licence Issued.
- Digital Licence Wallet: A page to display the user's digital Driving Licence card, including a QR code placeholder and essential licence details with a download option.
- Mock Learner Test: An interactive page featuring road sign question cards, multiple-choice options, and a score counter UI for users to practice learner's tests.

## Style Guidelines:

- The visual design emphasizes trust and clarity. The primary color is a professional and accessible medium blue (#2874BA). The background uses a heavily desaturated variant of the primary's hue for a clean, unobtrusive feel (#F1F5FA). A slightly more saturated and brighter blue serves as the accent color for interactive elements and call-to-actions (#4A90E2), ensuring prominence while maintaining visual harmony within the 'government-friendly' palette of blues, whites, and subtle greys.
- Body and headline font: 'Inter' (sans-serif) for its modern, clear, and highly readable qualities across all content types, supporting a citizen-friendly experience.
- All icons should be sourced from Bootstrap Icons, maintaining consistency and a clean aesthetic throughout the interface.
- The layout is strictly mobile-first and fully responsive, leveraging the Bootstrap grid system and Tailwind utility classes. Key elements include card-based UI, clean form layouts, modern wizard-style step indicators, and visual timelines, all designed with ample touch-friendly areas and soft shadows.
- Subtle animations are incorporated for UI interactions, such as progress step transitions, modal dialog appearances, and mock navigation, enhancing user engagement without being distracting.
