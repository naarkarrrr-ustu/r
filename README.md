## 📌 Overview

**Saathi Connect** is a UX redesign of the existing Sarathi Parivahan portal, aimed at transforming a complex government service into a **simple, guided, and user-friendly digital experience**.

This project was built as part of the **MOSAIC UI/UX Competition**, where participants were required to analyze a real-world product and redesign key user journeys to improve usability and accessibility. ([Unstop][1])

---

## 🎯 Problem Statement

The current Sarathi portal suffers from:

* ❌ Cluttered interface with too many scattered options
* ❌ Lack of guided workflow for new users
* ❌ No clear application progress tracking
* ❌ Confusing navigation and dropdown structure
* ❌ Poor mobile and accessibility experience

---

## 💡 Our Solution

We redesigned the experience with a **user-first approach**, focusing on:

* ✅ Guided step-by-step application flows
* ✅ Clean and structured service dashboard
* ✅ Separate flows for Learner Licence (LL) & Driving Licence (DL)
* ✅ Real-time application tracking system
* ✅ Simplified navigation with categorized dropdown menus
* ✅ Centralized document management

---

## 🔄 Key User Flows

### 🪪 Learner Licence (LL) Flow

```text
State Selection
→ Dashboard
→ Apply for LL
→ Personal Details
→ Address Details
→ Document Upload
→ Fee Payment
→ Test Slot Booking
→ Confirmation
```

---

### 🚘 Driving Licence (DL) Flow

```text
State Selection
→ Dashboard
→ Apply for DL
→ Enter LL Details
→ Vehicle Class Selection
→ Document Upload
→ Fee Payment
→ Driving Test Booking
→ Confirmation
```

---

### 📊 Application Tracking

```text
Submitted
↓
Documents Verified
↓
Fee Paid
↓
Test Scheduled
↓
Licence Generated
```
